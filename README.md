# taylorallen.dev

Personal site. Dark, monochrome, all-monospace. The whole page is a living ASCII density field — content emerges from it, not on top of it.

## Stack

- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS v4
- Monaspace (Neon + Xenon), self-hosted
- Deployed on Vercel

## Development

```bash
pnpm install
pnpm dev       # localhost:3000
pnpm build     # production build
pnpm typecheck # tsc --noEmit
pnpm lint      # eslint
pnpm test      # vitest
```
